# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version:

1. **Be respectful.** Disagree on technical decisions, not on people.
2. **Be welcoming.** First-time contributors get patience; experienced contributors set the tone.
3. **Be specific.** "This is wrong" without a reason isn't actionable. Cite the line, the test, the ADR.
4. **No harassment.** Personal attacks, sexualized language or imagery, deliberate intimidation, sustained disruption — any of these get you removed from the project.

## Scope

This applies in all project spaces: GitHub issues, PRs, discussions, the gist comment threads, and any Discord / Matrix / Slack channels that ever spin up.

## Enforcement

Report concerns to `ruv@ruv.net` (private). The project maintainer triages and acts — first response within 3 business days, action within 10. Actions range from a private warning to a permanent ban depending on severity and repetition.

You don't need to know the exact violated rule to report something that feels wrong. That's the maintainer's job to figure out.
